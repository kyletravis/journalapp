## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

## How to Use

1. Click **"+ New Entry"** to create a new journal entry
2. Enter a title for your entry
3. Write your thoughts in markdown format
4. Click **"Preview"** to see how your markdown will be rendered
5. Your entries are automatically saved to local storage
6. Select any entry from the sidebar to view or edit it
7. Hover over an entry and click the delete icon to remove it

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS v3** - Styling
- **React Markdown** - Markdown rendering
- **date-fns** - Date formatting

## Keyboard Shortcuts

- The editor auto-saves your work as you type (1 second delay)
- All your entries are stored locally in your browser
